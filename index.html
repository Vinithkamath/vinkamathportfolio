Projects

Project 1: PARKEASE (End-To-End Smart Parking)

The hardware system, built around an ESP32 microprocessor and ultrasonic sensors, ensures accurate vehicle detection at parking lot entrances and exits. The software component leverages optical character recognition (OCR) and computer vision for efficient license plate recognition and vehicle identification. The system includes a versatile payment solution with a wallet feature for registered users and UPI-based dynamic QR codes for unregistered users.

Technologies Used: Python,ESP32, OCR, Computer Vision, UPI, Ultrasonic Sensors

GitHub Repository

Project 2: Solar Based Smart Electric Vehicle Charging Station

In this project, I designed solar-powered EV charging stations using microcontrollers and the Arduino IDE. The system supports Wireless Power Transfer (WPT) and Conductive Charging (CC) to optimize energy collection and improve the overall EV charging process.

Technologies Used: Microcontrollers, Arduino IDE, Solar Power, Wireless Power Transfer

Project 3: DROP (Desalination of Water)

In the DROP project, I developed a sustainable hybrid desalination system that combines membrane filtration and distillation to purify water. The system integrates sensors like TDS, turbidity, temperature, and pH sensors, which are controlled by ESP32 and Raspberry Pi microcontrollers.

Technologies Used: ESP32, Raspberry Pi, RO Filters, Membrane Filtration, Firebase

GitHub Repository

Project 4: TRAFFIC MITRA

This project involved creating two separate applications, one for the public and another for traffic police, to streamline document verification processes. The public app allows users to upload or fetch key documents from DigiLocker, while the traffic police app uses a generated QR code to access and verify documents.

Technologies Used: Figma, DigiLocker, QR Code, Document Management

Figma Prototype

Project 5: Automated Sorting System for Industrial Processes (Internship)

This project involved creating an automated system to detect and sort color-based objects on a conveyor belt using Solid Edge, Raspberry Pi, Arduino, and OpenCV. The system used machine vision to detect objects.

Technologies Used: Raspberry Pi, Arduino, OpenCV, Solid Edge
